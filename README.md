# Evidence pojištěných - Java konzolová aplikace
Tento projekt je jednoduchá konzolová aplikace v Javě, která umožňuje spravovat seznam pojištěných osob.
Aplikace vznikla jako tréninkový projekt pro procvičení objektového programování, práce s třídami, seznamy a základní konzolovou interakcí.

---

# Funkce aplikace
- Přidání nové osoby
  -Uživatel zadá jméno, příjmení, věk a telefonní číslo.
